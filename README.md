# 🧩 Maze Data Structures

MazeData é um jogo educacional desenvolvido em React com o objetivo de ensinar Estruturas de Dados de forma prática, visual e interativa.

Ao longo da jornada, o jogador explora diferentes desafios baseados em estruturas fundamentais da computação, aprendendo seus conceitos por meio de mecânicas e situações contextualizadas em uma história.

## 🎯 Objetivo

Ensinar conceitos de Estruturas de Dados através da resolução de desafios interativos, permitindo que os estudantes compreendam o funcionamento das estruturas de maneira intuitiva e aplicada.

## 📚 Estruturas Abordadas

* 🏪 Fila (Queue)
* 🏰 Pilha (Stack)
* 🚪 Lista Encadeada (Linked List)
* 🌳 Árvore (Tree)
* 🕸️ Grafo (Graph)

## 🎮 Como Funciona

O jogador assume o papel de um viajante que precisa recuperar fragmentos do conhecimento espalhados pelo Reino MazeData.

Cada fase apresenta uma estrutura de dados diferente, onde o usuário realiza operações como:

* Inserção
* Busca
* Atualização
* Remoção
* Navegação entre elementos

Além disso, o jogo apresenta explicações, dicas e tutoriais guiados para auxiliar na aprendizagem.

## 🛠️ Tecnologias Utilizadas

* React
* JavaScript
* Vite
* CSS
* React Joyride
* React Hot Toast
* Framer Motion


Projeto desenvolvido como atividade acadêmica para auxiliar o ensino de Estruturas de Dados por meio da gamificação.
